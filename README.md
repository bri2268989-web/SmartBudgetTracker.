# SmartBudgetTracker.

A personal finance app built with Blazor WebAssembly.  
Track income, expenses, budget balance, and view detailed history with charts.

## Features
- Add, edit, and delete transactions
- Track income vs expenses
- Category filtering
- Date filtering
- LocalStorage persistence
- Installable PWA (works on mobile)
- Interactive charts

## Technologies
- Blazor WebAssembly
- C#
- HTML/CSS
- PWA Manifest + Service Worker
